## 关键设计决策

1. **异步架构**: 使用 Python asyncio 替代 Elixir OTP
2. **数据验证**: 使用 Pydantic 进行运行时类型验证
3. **模板引擎**: 使用 Jinja2 替代 Liquid
4. **HTTP 客户端**: 使用 httpx 替代内置 HTTP 库
5. **Agent 框架**: 计划使用 AgentScope 的底层组件
